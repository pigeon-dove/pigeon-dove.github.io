怪物砍法

题目

题目已经找不到了，只能口述。怪物有n滴血，你有k个技能，每个技能能打1, 2, 3, 4, …, k滴落血。假设每个技能都能用无限次，每个回合用一次技能，不限回合数。问恰好能把怪物打死的打法。

输入描述

输入两个正整数n和k，代表怪物的血量和技能的数量

1 <= n <= 1000

1 <= k <= 100

输出描述

一个整数，代表打法的数量对10^9+7取模的值。

样例输入

3 2

样例输出

3
